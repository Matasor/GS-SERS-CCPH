# Space Energy Monitor

## Global Solution - Soluções em Energias Renováveis e Sustentáveis
## Objetivo do projeto

O sistema analisa dados simulados de módulos operacionais de uma missão espacial, aplicando conceitos de energia, potência, fontes renováveis e sustentabilidade. A solução monitora temperatura, comunicação, energia disponível, potência consumida, captação de energia solar e status dos módulos.

## Funcionalidades

- Monitoramento de dados simulados da missão espacial.
- Exibição de temperatura, energia, potência, comunicação e energia renovável.
- Geração automática de alertas para situações críticas.
- Tomada de decisão básica com respostas automatizadas.
- Classificação do status dos módulos em NORMAL, ATENÇÃO ou CRÍTICO.
- Análise geral de sustentabilidade da missão.
- Relatório organizado no terminal.

## Lógica do sistema

O programa gera valores simulados para cada módulo da missão. Depois, analisa esses dados usando estruturas condicionais. Caso algum valor esteja fora do padrão seguro, o sistema gera alertas e apresenta uma decisão automatizada.

Exemplos de decisões:

- Ativar resfriamento emergencial.
- Reduzir consumo de energia.
- Usar baterias auxiliares.
- Acionar protocolo de comunicação reserva.
- Desligar cargas não essenciais.
